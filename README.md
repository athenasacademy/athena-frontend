# essentials

## Project setup

```
# yarn
yarn

# npm
npm install

# pnpm
pnpm install
```

```

# npm

# pnpm
pnpm dev
```

```

# npm
npm run build

# pnpm

```

### Lints and fixes files

```

yarn lint

# npm
npm run lint

# pnpm
pnpm lint
```
